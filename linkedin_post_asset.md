🚀 **Excited to share my latest project: SecureAssets Enterprise – A Next-Gen Cyber-Asset Management & Threat Intelligence Dashboard!** 🔐

Managing enterprise network assets while monitoring threat compliance is a massive challenge in modern cyber security. I built **SecureAssets** to bridge that gap with a unified, real-time threat response and asset registry interface.

Key Features & Capabilities:
* 📁 **Unified Asset Inventory:** Search, filter, and track lifecycle history with real-time audit event streams and dynamic QR tag generators.
* 🛡️ **Interactive Threat Matrix:** Track critical CVE vulnerabilities (like FreeRTOS stack overflows & remote access bypasses) mapped directly to affected live endpoints.
* 📈 **Live EDR Device Health:** High-frequency WebSocket telemetry monitoring real-time CPU, RAM, and network throughput charts.
* 🛰️ **Geo-Satellite Live Tracking:** Real-time physical device geolocations plotting coordinates with visual pulse nodes.
* 📄 **Security Rule Center:** Upload JSON health checks to audit compliance rules and automatically notify administrators of critical anomalies.
* 🔑 **Robust 2FA Security:** OTP integration generating QR setup keys for secure authenticator logins.

Built using:
💻 **React (Vite) | Node.js | Express | PostgreSQL | Socket.io | Tailwind CSS**
